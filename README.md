### 说明：

- https://github.com/ianisme/IanScrollView
- 这是利用UIScrollView实现的图片无限循环滚动的效果，在很多应用上都很常见。

### 功能如下：

- 1.获取网络图片
- 2.图片可以点击
- 3.控制是否显示pageControl
- 4.控制是否自动滚动
- 5.设置滚动时间
